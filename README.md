

#### Problem statement:
 - Build a web-based application for touch typing.

#### Technology Used:

 - HTML
 - CSS

### Features:
- WPM
- Accuracy
- Key Presses

 ## Install

To install all the dependencies of the project, run the following command:

    git clone https://github.com/shakti1590/Touch-typing.git
    npm install
    npm start
