## Chaabi Assignment

#### Problem statement:
 - Build a web-based application for touch typing.

#### Technology Used:
 - React Js
 - Java Script
 - HTML
 - CSS

### Features:
- WPM
- Accuracy
- Key Presses

 ## Install

To install all the dependences of the project, run the following command:

    git clone (https://github.com/shakti1590/ChaabiAssignment.git)
    npm install
    npm start
